# FOODFORTHEMOOD

## Production

Build the application for production:

```bash
docker compose up
```

## Development Server

Install dependencies: 

```bash
npm install
```

Start the development server on http://localhost:3000

```bash
npm run dev
```

### Run Tests:

```bash
npm run test
```

### Run Linter:

```bash
npm run lint
```

### Generate Documentation:

**Code docs:**
```bash
npm run doc
```

**Component docs:**
```bash
npm run story:dev
```
